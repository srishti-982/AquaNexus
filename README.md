# AquaNexus

🏆 Smart India Hackathon 2025

🌐 Live Demo: https://aqua-nexus.vercel.app/

AI-powered marine biodiversity monitoring platform using Raspberry Pi, YOLOv8, Gemini, and IPFS.

---

## 🚀 Features

- Real-time marine organism detection
- Automated organism counting
- Offline edge AI processing
- Raspberry Pi deployment
- YOLOv8 species classification
- Gemini-based validation
- Secure IPFS logging
- Plug-and-play microscope integration

---

## 🎯 Problem Statement

**SIH25043**

Embedded Intelligent Microscopy System for Identification and Counting of Microscopic Marine Organisms.
