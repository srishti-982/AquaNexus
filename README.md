# AquaNexus

🌐 Live Demo: https://aqua-nexus.vercel.app/

🏆 Smart India Hackathon 2025

AI-powered marine biodiversity monitoring platform using Raspberry Pi, YOLOv8, Gemini, and IPFS.


## Features
- Marine organism detection
- YOLOv8-based classification
- Raspberry Pi deployment
- Real-time counting
- Offline operation
- IPFS secure data logging
